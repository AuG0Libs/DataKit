includeReferenceAtKey:

Include the DKEntity that has a stored DKRelation at key.

- (void)includeReferenceAtKey:(NSString *)key

Parameters

key

The key to include. The object stored at key must be a DKRelation object.

Discussion

Include the DKEntity that has a stored DKRelation at key.

This is similar to a JOIN in a RDBMS.

Important: This operation might impact the query performance quite significantly if the query result set is large!
