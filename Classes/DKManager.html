dropDatabase:error:

Drops the specified database synchronously

+ (BOOL)dropDatabase:(NSString *)dbName error:(NSError **)error

Parameters

dbName

The name of the database to drop

error

The error object set on error

Return Value

YES on success, otherwise NO

Discussion

Drops the specified database synchronously

This feature must be explicitly enabled in the server configuration to make accidental or malicious deletion attempts harder.

'allowDrop': true

Warning: Use with extreme caution. This will permanently drop the specified database and all entity records.
